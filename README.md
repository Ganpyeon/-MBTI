# 🏴‍☠️항해 MBTI

요즈음 모두의 관심사인 MBTI검사를 항해 버전 리뉴얼을 통해, 항해 선원들 만의 MBTI를 검사할 수 있게 한다.

## 📆개발 기간

2023년 01월 16일 ~ 2023년 01월 17일

## 👥팀원

### FrontEnd

`박정현` : 메인 화면, 결과화면, css <br>
`전찬열` : 테스팅 화면 , css <br>

### Backend

`이승렬` : 아이디어, 결과도출 Alg. 작성 , 배포 <br>
`조민성` : API 작성 <br>

## 🛠️기술 스택

- HTML
- CSS
- JavaScript
- Python
- Flask
- mongoDB

## 💡API 구현 기능

- 닉네임 저장
- 질문 결과값 저장
- 결과내용 보기
- 결과내용을 닉네임을 통해서 다시보기
